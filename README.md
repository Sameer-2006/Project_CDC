# Project_CDC

This repository contains a conceptual prototype website for the **Career Development Centre (CDC), IIT (ISM) Dhanbad**.

The project was developed as part of a comparative study on Training & Placement practices across leading institutions, with the objective of demonstrating how improved transparency, recruiter engagement, and structured communication can be implemented through a centralized CDC portal.
